## Builds Editor v1.0

![Builds workspace](https://firebasestorage.googleapis.com/v0/b/codedex-io.appspot.com/o/blogs%2F2026%2Fcodedex-builds-v1%2Fbuilds.gif?alt=media&token=1406b4c5-7492-41cc-b8dd-6c0f9138a969)
<Caption text="OMG. It's just so... beautiful. 😍" />

Meet your new online code editor... ✨**[Codédex Builds](https://codedex.io/builds)**✨.

Codédex Builds has been around for a bit, but until now, a "Build" meant one HTML file, one CSS file, and one JavaScript file glued together inline. Fine for a quick demo. Not fine the moment you wanted folders, multiple stylesheets, image assets, or anything resembling a real project.

This was a _massive_ undertaking on the engineering side – they built out the infrastructure, leaderboards, and test cases for both Python and JavaScript. It was _a lot_, but we did it!

Here are the newest features:

- 📁 Multi-file projects
- 🖼️ Asset uploads (images, videos, audio, text files)
- ⚡ Live preview
- 🌐 Public URLs
- 🎨 Custom subdomains
- 🔀 Remix

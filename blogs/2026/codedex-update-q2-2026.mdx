## Builds v1.0

![Builds workspace](https://firebasestorage.googleapis.com/v0/b/codedex-io.appspot.com/o/blogs%2F2026%2Fcodedex-builds-v1%2Fbuilds.gif?alt=media&token=1406b4c5-7492-41cc-b8dd-6c0f9138a969)
<Caption text="OMG. It's just so... beautiful. 😍" />

Meet your new online code editor... ✨Codédex Builds✨.

Codédex Builds has been around for a while, but until now, a "Build" meant one HTML file, one CSS file, and one JavaScript file glued together inline. Fine for a quick demo. Not fine the moment you wanted folders, multiple stylesheets, image assets, or anything resembling a real project.

That changes with this update.

We rolled out ✨Builds v1.0✨: a real multi-file online code editor with public hosting baked in.


We're SOOO excited to share that we tested out a new Codédex feature: ✨**[Daily Challenges](https://codedex.io/daily-challenge)**✨. 

In March, we invited our entire community to solve **one coding problem a day**. Think Wordle. Think Daily Crossword. But for technologists lvling up together. There’s something incredibly special about this format: everyone wakes up to the same challenge. A shared ritual.

This was also a _massive_ undertaking on the engineering side – they built out the infrastructure, leaderboards, and test cases for both Python and JavaScript. It was _a lot_, but we did it!

Here are the newest features:

- 📁 Multi-file projects
- 🖼️ Asset uploads (images, videos, audio, text files)
- ⚡ Live preview
- 🌐 Public URLs
- 🎨 Custom subdomains
- 🔀 Remix

**GitHub Repo:** https://github.com/codedex-io/daily-challenges
